# 🧭 Smart Career Navigator
Smart Career Navigator is an interactive desktop application built using Java Swing that helps users explore career paths through a guided question-based system.
The application asks the user a series of carefully designed Yes/No questions and, based on their answers, guides them toward a suggested career direction.
## Project Idea
Choosing a career can be confusing.
Smart Career Navigator simplifies this process by:
- Asking structured personality and interest-based questions
- Analyzing user responses
- Suggesting a suitable career path
It works as a simple decision-based navigation system that simulates career guidance in an interactive and user-friendly way.
## How It Works
The application starts with a Welcome Screen.
#### When the user clicks Start:
- The welcome screen disappears.
- The first question appears.
- The user answers using Yes / No buttons.
- The system tracks responses internally.
- At the end of the questions:
- The application suggests a career path based on the collected answers
- The logic works like a simplified decision tree built using conditional statements.
## Features
- Modern UI with Gradient Background
- Interactive Welcome Screen
- Step-by-step Question Navigation
- Yes / No Button Interaction
- Response-Based Career Suggestion
- Smooth Screen Transition
- Decision-based logic implementation
- Built entirely using Java Swing
## Technologies Used

- Java Swing
- AWT (Graphics & Layout Management)
